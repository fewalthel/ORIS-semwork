# ORIS-semwork
<h3>Семестровая работа по предмету Основы разработки информационных систем (сайт на сервлетах)</h3>

Сайт представляет собой онлайн форум для взаимопомощи друг другу по учебе.

Есть личный кабинет с юзернеймом и статистикой по заданным вопросам и данным ответам.
Есть раздел «вопросы», куда пользователи могут писать свои вопросы по темам, а в комментариях под вопросом им могут отвечать другие пользователи.

Также есть роли: пользователь и админ. Админ может управлять другими пользователями, блокировать их учетные записи, удалять посты или ответы других пользователей. Пользовчатель может просматривать уже добавленные на сайт вопросы и ответы к ним, писать вопросы или ответы, менять свой юзернейм.

Существует система вознаграждений. Пользователи, которые помогли другим людям какое-то количество раз, получают вознаграждения, которые можно просмотреть в личном кабинете.
