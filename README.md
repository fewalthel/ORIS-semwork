# ORIS-semwork
<h3>Семестровая работа по предмету Основы разработки информационных систем (сайт на сервлетах)</h3>

Сайт представляет собой онлайн форум для взаимопомощи друг другу по учебе.

Есть личный кабинет с юзернеймом и статистикой по заданным вопросам и данным ответам.
Есть раздел «вопросы», куда пользователи могут писать свои вопросы по темам, а в комментариях под вопросом им могут отвечать другие пользователи.

Пользователь может просматривать уже добавленные на сайт вопросы и ответы к ним, писать вопросы или ответы, менять свой юзернейм и пароль,почту менять не может.
